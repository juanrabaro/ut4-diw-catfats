# ut4-diw-catfats
En este archivo se describirán los cambios del proyecto desde la apliación figma hasta la página final maquetada al completo:

En la versión para PC:
En el pie de página se ha añadido el botón para cambiar la página a modo oscuro.

En la versión para dispositivos móviles:
En el encabezado se ha sustituido los enlaces a las otras páginas por un menú hamburguesa desplegable, en el pie de página se ha añadido el botón de modo oscuro y los iconos de redes sociales, la infomación de redes sociales están uno al lado de otro en vez de debajo.
 Se ha sustituido en la página index la wiki por el flujo de usuario para hacer más fácil al usuario encontrar este apartado ya que en la versión para PC se encuentra en esta página, por tanto en la página donde se encontraba el flujo de usuario está la wiki.
